# Game Spec

이 파일을 수정하면 Claude가 읽고 게임에 반영합니다.

---

## 캔버스 & 레이아웃

- 해상도: 390 × 693 (9:16 고정)
- CSS로 뷰포트에 맞게 스케일, 비율 유지
- 배경색: `#0a0a0f`
- 외부 배경색: `#111`

---

## 플레이어

- 반지름: 18
- 색상: `#00e5ff` (네온 시안)
- 초기 위치: 화면 하단 중앙 (x: GAME_W/2, y: GAME_H - 80)
- 조작: 터치 / 마우스 X 위치 추적 (Y 고정)

---

## 총알

- 반지름: 3
- 색상: `#ffe600` (네온 옐로우)
- 초기 속도: 5 (위쪽)
- 초기 발사 간격: 650ms (자동)
- 멀티샷 시: 중앙 + 좌우 spread (SHOT_SPREAD 12도 간격)

---

## 적

### 일반 적
- 반지름: 14
- 색상: `#ff2d55` (네온 레드)
- 속도: 2
- 스폰 간격: 1300ms

### 빠른 적 (fast enemy)
- 스폰 확률: 25%
- 반지름: 10 (일반보다 4 작음)
- 색상: `#ff9f0a` (네온 오렌지)
- 속도: 5

### 호밍 적 (homing enemy)
- 스폰 확률: ~5% (rand >= 0.85 && rand < 0.90)
- 반지름: 14 (일반과 동일)
- 색상: `#bf5af2` (퍼플)
- 속도: 2.5
- 이동: 매 프레임 플레이어 방향으로 정규화 벡터 이동

### 슈터 적 (shooter enemy)
- 스폰 확률: 10% (rand >= 0.90)
- 반지름: 12
- 색상: `#30d158` (네온 그린)
- 속도: 1.2 (느리게 직진)
- 공격: 2.5초마다 플레이어 방향으로 미사일 발사
- 미사일: 반지름 5, 속도 4, 색상 동일 (네온 그린), 방어막 무시

---

## XP & 레벨

- 적 처치 1마리 = XP +1
- 레벨업 필요 XP: 4에서 시작, 레벨업 시 × 1.3 (정수 내림)
- 레벨업 시 게임 일시정지 → 카드 3장 선택 화면

---

## 스테이지

- 12초마다 스테이지 +1
- 속도 스케일: 스테이지당 +2.5% (`1 + (s-1) * 0.025`)
- HP 스케일: 4스테이지마다 +1 HP (`1 + floor(s/4)`)

## 웨이브

스테이지는 3단계 페이즈로 진행: `playing → warning → wave → playing → …`

| 페이즈 | 시간 | 동작 |
|---|---|---|
| playing | 12s | 일반 트리클 스폰 (스테이지당 -40ms, 최소 600ms) |
| warning | 2s | 트리클 중단, "WARNING / WAVE INCOMING" 오버레이 표시 |
| wave | 최대 5s | `15 + floor((stage-1)/2)`마리를 3초에 걸쳐 빠른 간격으로 스폰 |

- 웨이브 적은 `isWave: true` 플래그 — 모두 처치 시 즉시 다음 페이즈로
- 5초 내 미처치 시 자동으로 다음 페이즈 진행
- 웨이브 시작 시: 스테이지 +1, "STAGE {n}" 52px 스케일 인/아웃 애니메이션
- 캔버스 가장자리 주황색 플래시 (`rgba(255,160,0,a)`, 두께 28, decay 0.035)

## 업그레이드 카드

레벨업 시 아래 목록에서 무작위 3장 제시. 1장 선택.

| 이름 | 효과 |
|---|---|
| Rapid Fire | 발사 간격 -50ms (최소 100ms) |
| Swift Bullets | 총알 속도 +2 |
| Power Up | 총알 파워 +0.5 |
| Shield | 쉴드 활성화 (반지름 player.r + 14, 파워 1) |
| Shield Range | 쉴드 반지름 +8 |
| Shield Power | 쉴드 파워 +0.5 |
| Add Shot | 발사 수 +1 (spread) |
| Left Wing | 왼쪽 비행기 추가 → 레이저 빔 (0.4 dmg/s, 가장 가까운 적 조준) |
| Right Wing | 오른쪽 비행기 추가 → 유도탄 (2초마다 가장 가까운 적 추적, 공격력 2) |
| Left Wing+ | 레이저 공격력 +0.2/s |
| Right Wing+ | 유도탄 공격력 +1 |

## 히트 이펙트

적 처치 시 해당 위치에 생성:

| 종류 | 내용 |
|---|---|
| ring | 바깥으로 퍼지는 원형 테두리, 주황 (`rgba(255,120,50,a)`) |
| particle | 6방향으로 흩어지는 점, 노랑 (`rgba(255,200,80,a)`) |
| text | `+1` 위로 떠오르며 페이드아웃, 옐로우 |
| hit | 피격(미처치) 시 흰색 원형 플래시 (`rgba(255,255,255,a)`) |

- ring/particle/text 수명: 약 0.3초 (life 1.0 → 0, 매 프레임 -0.055)
- hit 수명: 약 0.18초 (매 프레임 -0.18)

---

## 점수

- 매 프레임 내부 카운터 +1, 60프레임마다 표시 점수 +1 (≈ 초당 1점)
- HUD 좌상단 및 게임 오버 화면에 표시

---

## HUD

- 좌상단: `SCORE: {n}` (시안, 18px 모노스페이스)
- 우상단: `LV {n}` (퍼플, 14px 모노스페이스)
- 상단 XP 바: 보라 (`#b388ff`), 높이 5px
- 스테이지 전환 시 중앙 오버레이: `STAGE {n}` (옐로우 `#ffe600`, 22px, 2.2초 표시 후 사라짐, 레벨업 중에는 숨김)

---

## 시작 화면

- 최초 접속 시 게임 루프/스폰 비활성 상태로 대기
- 오버레이: 타이틀("VOID / SURVIVOR") + START 버튼
- START 버튼 클릭 시 게임 시작
- 재시작(RESTART) 시에는 시작 화면 없이 바로 게임 진입

---

## 게임 오버

- 적이 플레이어에 닿으면 즉시 게임 오버
- 오버레이: "GAME OVER" + 최종 점수 + RESTART 버튼
