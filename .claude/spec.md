# Game Spec

이 파일을 수정하면 Claude가 읽고 게임에 반영합니다.

---

## 캔버스 & 레이아웃

- 해상도: 390 × 693 (9:16 고정)
- CSS로 뷰포트에 맞게 스케일, 비율 유지
- 배경색: `#0a0a0f`
- 외부 배경색: `#111`

---

## 플레이어

- 반지름: 18
- 색상: `#00e5ff` (네온 시안)
- 초기 위치: 화면 하단 중앙 (x: GAME_W/2, y: GAME_H - 80)
- 조작: 터치 / 마우스 X 위치 추적 (Y 고정)

---

## 총알

- 반지름: 3
- 색상: `#ffe600` (네온 옐로우)
- 속도: 6 (위쪽)
- 발사 간격: 400ms (자동)
- 멀티샷 시: 중앙 + 좌우 14px 오프셋 3발

---

## 적

### 일반 적
- 반지름: 14
- 색상: `#ff2d55` (네온 레드)
- 속도: 2
- 스폰 간격: 1200ms

### 빠른 적 (fast enemy)
- 스폰 확률: 25%
- 반지름: 10 (일반보다 4 작음)
- 색상: `#ff9f0a` (네온 오렌지)
- 속도: 5

### 호밍 적 (homing enemy)
- 스폰 확률: 15% (rand >= 0.85)
- 반지름: 14 (일반과 동일)
- 색상: `#bf5af2` (퍼플)
- 속도: 2.5
- 이동: 매 프레임 플레이어 방향으로 정규화 벡터 이동

---

## XP & 레벨

- 적 처치 1마리 = XP +1
- 레벨업 필요 XP: 5에서 시작, 레벨업 시 × 1.5 (정수 내림)
- 레벨업 시 게임 일시정지 → 카드 3장 선택 화면

---

## 업그레이드 카드

레벨업 시 아래 목록에서 무작위 3장 제시. 1장 선택.

| 이름 | 효과 |
|---|---|
| Rapid Fire | 발사 간격 -100ms (최소 100ms) |
| Swift Bullets | 총알 속도 +3 |
| Big Shot | 총알 반지름 +4 |
| Wide Guard | 플레이어 반지름 +8 |
| Double Shot | 멀티샷 활성화 (3발 동시 발사) |

---

## 히트 이펙트

적 처치 시 해당 위치에 생성:

| 종류 | 내용 |
|---|---|
| ring | 바깥으로 퍼지는 원형 테두리, 주황 (`rgba(255,120,50,a)`) |
| particle | 6방향으로 흩어지는 점, 노랑 (`rgba(255,200,80,a)`) |
| text | `+1` 위로 떠오르며 페이드아웃, 옐로우 |

- 이펙트 수명: 약 0.3초 (life 1.0 → 0, 매 프레임 -0.055)

---

## 점수

- 매 60프레임마다 +1 (생존 시간 기반)
- 게임 오버 화면에 최종 점수 표시

---

## HUD

- 좌상단: `SCORE: {n}` (시안, 18px 모노스페이스)
- 우상단: `LV {n}` (퍼플, 14px 모노스페이스)
- 상단 XP 바: 보라 (`#b388ff`), 높이 5px

---

## 시작 화면

- 최초 접속 시 게임 루프/스폰 비활성 상태로 대기
- 오버레이: 타이틀("VOID / SURVIVOR") + START 버튼
- START 버튼 클릭 시 게임 시작
- 재시작(RESTART) 시에는 시작 화면 없이 바로 게임 진입

---

## 게임 오버

- 적이 플레이어에 닿으면 즉시 게임 오버
- 오버레이: "GAME OVER" + 최종 점수 + RESTART 버튼
