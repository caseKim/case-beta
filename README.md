# Case Beta

모바일 캐주얼 디펜스 게임. 플레이어가 자동으로 총알을 발사해 위에서 내려오는 적을 처치하고, 레벨업 카드로 능력을 강화한다.

## 플레이 방법

- **이동**: 화면을 터치하거나 마우스를 움직여 플레이어 이동
- **공격**: 자동 발사
- **적이 플레이어에 닿으면 게임 오버**
- **적 처치 → XP 획득 → 레벨업 시 카드 3장 중 1개 선택**

## 실행

```bash
npm install
npm run dev
```

## 빌드

```bash
npm run build
```

## 기술 스택

- React 18 + Vite
- Canvas API (no game library)
- 9:16 고정 해상도 (390×693), CSS 스케일링으로 모든 기기 동일 난이도
