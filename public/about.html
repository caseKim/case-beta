Void Survivor

case-beta-kappa.vercel.app

게임 소개

Void Survivor는 브라우저에서 바로 즐기는 아케이드 슈팅 게임입니다. 끊임없이 쏟아지는 적들을 피하고 처치하며 최대한 오래 생존하세요. 스테이지가 올라갈수록 더 강하고 다양한 적이 등장합니다.

레벨업 시 무작위 업그레이드 카드를 선택해 자신만의 빌드를 만들고, 글로벌 리더보드에서 다른 플레이어와 점수를 겨뤄보세요.

기술 스택

React + HTML Canvas로 제작된 싱글 페이지 웹앱입니다. 사용자 인증 및 리더보드 데이터는 Google Firebase(Firestore)로 관리됩니다.
